# Personal Development Blog

Astro 기반의 정적 개인 개발 블로그 MVP입니다. 한국어 콘텐츠를 기본으로 하며, 게시글과 프로젝트는 로컬 Markdown/MDX 컬렉션으로 관리합니다.

## Commands

```bash
npm ci
npm run dev
npm run build
npm run check
npm run lint
npm run format:check
npm run test:unit
```

초기 프로덕션 콘텐츠에는 게시글을 포함하지 않습니다. 확인된 게시글이 `src/content/posts/`에 추가되면 홈의 카테고리 필터와 상세 라우트가 자동으로 생성됩니다.
