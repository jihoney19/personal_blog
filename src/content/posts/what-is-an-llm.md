LLM은 Large Language Model의 줄임말로, 많은 텍스트 데이터를 학습해 언어의 패턴을 다루는 모델입니다. 질문에 답하거나 문장을 요약하고, 주어진 형식에 맞춰 새로운 텍스트를 생성할 수 있습니다.

## LLM은 무엇을 학습할까

LLM은 문장 안에서 단어와 토큰이 어떤 순서로 나타나는지 학습합니다. 문맥을 바탕으로 다음에 올 가능성이 높은 토큰을 예측하는 과정을 반복하면서 문법, 표현, 지식의 패턴을 익힙니다.

학습 데이터의 양이 많고 모델의 규모가 커질수록 복잡한 문맥을 다루는 능력이 좋아질 수 있지만, 모델의 답변이 항상 사실이라는 뜻은 아닙니다.

## 질문에 답하는 과정

사용자가 입력한 문장은 토큰으로 나뉘어 모델에 전달됩니다. 모델은 입력된 토큰의 관계를 계산하고, 다음 토큰 후보의 확률을 구합니다. 이 과정을 여러 번 반복해 하나의 답변을 만듭니다.

```text
질문 입력 -> 토큰화 -> 다음 토큰 예측 -> 답변 생성
```

## LLM의 한계

LLM은 데이터베이스처럼 사실을 조회하는 도구가 아닙니다. 학습 시점 이후의 정보가 없을 수 있고, 그럴듯하지만 틀린 내용을 생성할 수도 있습니다. 따라서 중요한 정보는 공식 문서나 원본 자료로 다시 확인해야 합니다.

또한 질문의 표현이 모호하면 의도와 다른 답변을 만들 수 있습니다. 원하는 형식, 범위, 참고할 자료를 구체적으로 적을수록 결과를 검토하기 쉬워집니다.

## 정리

LLM은 언어 패턴을 학습해 다음 토큰을 예측하는 모델입니다. 자연스러운 문장을 만드는 데 강하지만, 답변을 사실로 받아들이기보다 검토가 필요한 생성 결과로 이해하는 것이 중요합니다.
